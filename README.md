# PharmaMind AI

AI-powered drug information assistant for pharmacy students and pharmacists.

## Features

- Drug monograph generation
- Drug indications
- Mechanism of action
- Dosing information
- Contraindications
- Drug interactions
- Side effects
- Pregnancy & lactation information
- Counseling points
- Exam-focused notes

## Tech Stack

- Python
- Streamlit
- Ollama
- Llama 3.2
- RapidFuzz

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
